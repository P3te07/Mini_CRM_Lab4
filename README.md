# Mini-CRM

O aplicație simplă de tip Customer Relationship Management,
dezvoltată cu HTML, CSS și JavaScript vanilla.

## Descriere

Mini-CRM permite utilizatorilor să:

- Adauge și gestioneze clienți/contacte
- Vizualizeze statusul fiecărui client (Lead / Activ / Inactiv)
- Caute și filtreze clienții după nume sau status
- Editeze și șteargă înregistrări
- Vizualizeze statistici sumare (total clienți, activi, leads)

## Tehnologii utilizate

- HTML5
- CSS3 (Flexbox, Grid, animații)
- JavaScript ES6+
- LocalStorage pentru persistența datelor

## Instrucțiuni de rulare

1. Clonează repository-ul:

```bash
   git clone https://github.com/P3te07/Mini_CRM_Lab4
```

2. Navighează în folder:

```bash
   cd mini-crm
```

3. Deschide `src/index.html` în browser.

Nu sunt necesare dependențe sau instalări suplimentare.
